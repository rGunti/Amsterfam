# ADR-002 — Angular PWA over Blazor

**Status:** Accepted  
**Date:** 2026-06-06

## Decision

Use Angular as a Progressive Web App for the frontend.

## Reasons

- Owner has strong existing Angular experience
- PWA gives cross-platform support (mobile, desktop) without native app overhead

## Rejected Alternative

**Blazor** — no real-project experience; switching costs outweigh any benefits.
